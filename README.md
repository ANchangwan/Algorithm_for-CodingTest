<h1><li>Gready Algorithm</li></h1>
<p>그리디 알고리즘은 말 그대로 탐욕법이다. 다시 말해 <strong> 현재 상황에서 지금 당장 좋은것만 고르는 방법 </strong>을 의미한다. 그리디 알고리즘을 이용하면 매 순간 가장 좋아 보이는 것을 선택하며, 현재의 선택이 나중에 미칠 영향에 대해서는 고려하지 않는다. <strong>'가장 큰 순서대로''가장 작은 순서대로</strong>와 같은 기준을 제시해준다. 대체로 이 기준은 <strong>정렬알고리즘<strong>을 사용했을 때 만족시킬 수 있으므로 그리디 알고리즘 문제는 자주 정렬 알고리즘과 짝을 이뤄 출제된다.</p>
